## Best Starting Schedule
Use this for 30 days before increasing volume.

Daily target:
- YouTube Shorts: 2 posts/day
- Instagram Reels: 2 posts/day
- Facebook Page/Reels: 1 post/day
- Facebook Groups: 3–4 posts/week, not every day in the same group

Do not post the exact same video repeatedly. Rotate hooks and product angles.

## Posting Rules
1. Use a different hook each time.
2. Send every post to the product landing page, not Amazon.
3. Use UTM links per platform.
4. Pin the best-performing Reel/Short each week where possible.
5. Reuse winning hooks, but remake the video with a different opening visual.
6. Check weekly: views, clicks, checkout starts, purchases.
