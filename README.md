# natureswaysoil/video

**AI-Powered Automated Video Marketing Engine for Nature's Way Soil**

Turns your Google Sheet product catalog into scroll-stopping videos using:
- OpenAI (smart scripts)
- HeyGen (avatars)
- **Pexels API** (B-roll — your key is already added!)
- Real farm footage (we'll add this next)

Auto-posts to Instagram, X, Pinterest & YouTube. Runs twice daily on Google Cloud Run.
